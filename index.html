SOHO Network Design

Objective: Designed and deployed a secure, high-performance Small Office/Home Office (SOHO) network with optimized performance and robust security protocols.

Network Hardware Configuration:

  • Routers and Switches: Utilized Ubiquiti routers and Layer 2/3 managed switches from the UniFi product line for scalability and high performance.
  • VLANs: Established VLANs to segment network traffic based on device type and function.
  • Wireless Access Points (WAPs): Deployed dual-band UniFi WAPs supporting both 2.4 GHz and 5 GHz frequencies for optimal wireless coverage.

Security Implementations:

  • Next-Generation Firewall (NGFW): Implemented a pfSense-based NGFW featuring deep packet inspection (DPI), application-layer filtering, and real-time intrusion prevention capabilities.
  • Intrusion Detection/Prevention Systems (IDS/IPS): Integrated Snort IDS/IPS within the pfSense environment to monitor and block malicious activities.
  • VPN Configuration: Established secure site-to-site and remote access VPN tunnels using OpenVPN, fortified with AES-256 encryption.
  • DNS Sinkhole: Configured pfBlockerNG on pfSense to act as a DNS sinkhole, preventing access to known malicious domains.

Results:

Successfully implemented a secure SOHO network capable of mitigating advanced threats while optimizing performance for business operations.
